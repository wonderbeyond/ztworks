# ztWorks(专题工厂)

互联网专题制作流程化、工程化实践。Enjoy it!

---

## 准备工作

- 安装nodejs（省略）

- 安装依赖模块

  ```
  npm install
  ```

## 工作流

参考`zts/`下面的专题样板开始制作专题

```
制作。。。
```

执行gulp查看构建命令
